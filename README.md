# BlackYC Game

## Name
"BlackYC" Card Game

## Description
It is a card game based on Blackjack that enables multiple users to play simultaneously and track their game records.

## Technology Stack
* UI: TypeScript, Vue.js, and Bootstrap
* Server: Node.js, Socket.IO,and MongoDB
